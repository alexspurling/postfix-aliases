Allows you to manage postfix email aliases.

== Description ==
This plugin allows you to add and remove email aliases for use by the postfix MTA (mail transfer agent). Aliases are maintained in a mysql table which must be then be read by postfix. This means you can set up mail forwarding directly from your blog.
